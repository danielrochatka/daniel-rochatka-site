Vocarum is a framework and prototype system for consensus-based journalism that preserves the structure of evidence rather than compressing it into a single authoritative narrative.

The core design problem: when multiple sources address the same event or question, there is always some overlap and some divergence. A useful information system should make both visible. Vocarum uses multiple independent AI models to interpret the same source material, then maps where interpretations converge and where they diverge in ways that can be classified and reasoned about — factual disagreement, methodological difference, temporal scope, definitional conflict.

The architecture includes source aggregation, claim extraction, multi-model interpretation, consensus mapping, divergence classification, and versioned publication with manifest-based provenance tracking.

The accompanying research paper is published at [doi.org/10.5281/zenodo.16636108](https://doi.org/10.5281/zenodo.16636108). Further detail is on the [Procyonsoft project page](https://procyonsoft.com/projects/vocarum/).
